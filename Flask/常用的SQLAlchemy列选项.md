

常用的SQLAlchemy列选项

| 选项名      | 说明                                                       |      |
| ----------- | ---------------------------------------------------------- | ---- |
| primary_key | 如果设为rue,列为表的主键                                   |      |
| unique      | 如果设为True,列不允许出现重复的值                          |      |
| index       | 如果设为True,为列创建索引,提升查询效率                     |      |
| nullable    | 如果设为True,列允许使用空值如果设为 False,列不允许使用空值 |      |
| default     | 为列定义默认值                                             |      |
