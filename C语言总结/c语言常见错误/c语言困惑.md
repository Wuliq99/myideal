```c
#include<stdio.h>
int main(){
	int a[3][4]={{1},{3,4,5},{6,7,8,9}};
	printf("%d\n",*(a+1)[1]);
	printf("%d\n",(*(a+1))[1]);
}
/*结果：
6
4
*/
/*
		[ ]的优先级高于*，和()的优先级同级。
		对不加括号的*(a+1)[1]，要先作[1]后求*。a的类型是int (*)[4]，a+1 就是 a[1]行，再作(a+1)[1]，
		即这行的下一行，相当于原数组的a[2](但不是(a+1)+1)。这时再求 * ，即求*(a[2]+0) 或 **(a+2)得6。



*/
```
