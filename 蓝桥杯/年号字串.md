## 题目

> [年号字串](https://www.lanqiao.cn/problems/605/learning/)

## 题目描述

**本题为填空题，只需要算出结果后，在代码中使用输出语句将所填结果输出即可。**

小明用字母 A 对应数字 1，B 对应 2，以此类推，用 Z 对应 26。对于 27 以上的数字，小明用两位或更长位的字符串来对应，例如 AA 对应 27，AB 对应 28，AZ 对应 52，LQ 对应 329。

请问 2019 对应的字符串是什么？

### 运行限制

- 最大运行时间：1s
- 最大运行内存: 128M

## 题目分析

​		由题意可得，字符串只能由 A 到 Z 这 26 个大写字母组成，而由题目描述可得，在一定程度上满足 26 进制（26个字母对应26种状态），因此可用**除k取余法**将 2019 表示成 26 进制数。可得答案为 BYQ。

## 代码实现

注意：此结果为逆序，填写结果时需要转为正序

```c
#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[]) {
    int sum = 2019;
    while (sum) {
        int temp = sum % 26;
        sum /= 26;
        printf("%c", 'A' + temp - 1);
    }
    return 0;
}
```
